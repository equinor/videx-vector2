Installation

npm install --save @equinor/videx-vector2

Usage

// ES6
import Vector2 from '@equinor/videx-vector2';

// ObservableHQ
Vector2 = require('@equinor/videx-vector2@X.X.X');

Where X.X.X is desired version number.
